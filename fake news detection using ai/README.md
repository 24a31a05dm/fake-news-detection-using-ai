# VeriSight AI

A clean, responsive fake news detection website built as a static front-end project.

## Features

- Clear hero and analyzer layout
- Smooth reveal and ambient animations
- Client-side credibility scoring demo
- Positive and risk signal summaries
- Mobile-friendly responsive design

## Files

- `index.html` - page structure
- `styles.css` - styling and animations
- `script.js` - interactive fake news analysis logic

## Usage

Open `index.html` in a browser to use the site locally.
